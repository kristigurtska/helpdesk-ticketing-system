# IT Helpdesk Ticketing System

A simple **IT Helpdesk Ticketing System** built with **HTML, CSS, and JavaScript**.  
Users can create tickets, change their status (Open → In Progress → Resolved), and delete them. All data is saved in the browser using `localStorage`.

---

##  Live Demo

Check out the live demo here:  
[Live Demo](https://kristigurtska.github.io/helpdesk-ticketing-system/)

---

##  Features

- Create new tickets with title, description, and priority
- Change ticket status: **Open → In Progress → Resolved → Open**
- Delete tickets
- Data persistence with `localStorage`


---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Browser `localStorage`
